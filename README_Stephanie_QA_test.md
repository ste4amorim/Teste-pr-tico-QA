# QA Practical Test

This repository contains the report of a practical test for a Junior QA Analyst position.

The test consisted of analyzing two web pages and identifying possible issues related to:

- functionality
- navigation
- forms
- content consistency
- usability

The analysis was performed through exploratory testing of the pages.
